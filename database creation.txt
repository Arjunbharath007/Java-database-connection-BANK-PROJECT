
This project includes a database named "proj."
Within this database, there are two tables: "bank" and "details."
-----------------------------------------------------------------
Table "bank":

Column 1: "id" (Primary key, Auto Increment)
Column 2: "maxid"
Column 3: "amount"
------------------------------------------------------------------
Table "details":

Column 1: "id" (Primary key, Auto Increment)
Column 2: "name"
Column 3: "email"
Column 4: "password"
------------------------------------------------------------------
Once these tables are created in the "proj" database,
the project is ready for coding.
